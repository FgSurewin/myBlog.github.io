The Pointer And Class in C++

Posted on 2020-05-11 In programming Valine:

不同的语言, 不同语法, 太容易混乱了. 基于此特意写下自己容易疑惑的几个点.

Reference

& 的两种意思

  • 声明地址的操作符
  • 声明引用的操作符

引用的意思是指 alias, 仅仅是一个别名, 但共同使用同一个地址

Pointer

“*“ 的两种意思

  • 声明指针的操作符
  • dereference 的作用

数组的奥秘

数组的中变量实际上是存放着第一个项的地址, 由于是用int去定义的, 所以每次加 1 都会加 4 个 bit 从而形成新的地址

Class

大体思想与 Java 相同, 但是应该注意以下问题

  • 顺序问题
  • 变量权限问题
  • 声明 method 的方式
    • 内部声明
    • 外部声明
